ネイティブ <select> vs Select-only Combobox

ドロップダウンの中身をカスタマイズできるかどうかの違い
1. ネイティブ <select> + CSS
Sample

トリガーの見た目は変えられるが、ドロップダウンの中身はブラウザ任せ

実装イメージ
<select>
  <option>りんご</option>
  <option>みかん</option>
  <option>ぶどう</option>
</select>

<!-- JS 不要 -->
<!-- ARIA 不要（ブラウザが自動付与） -->
<!-- キーボード操作も自動 -->
2. Select-only Combobox
Sample
選択してください ▼
  • 🍎 りんご青森県産 / さっぱり甘い
  • 🍊 みかん愛媛県産 / ジューシー
  • 🍇 ぶどう山梨県産 / 濃厚な甘み
  • 🍌 バナナフィリピン産 / 栄養たっぷり
  • 🍓 いちご栃木県産 / とちおとめ

ドロップダウンの中身まで自由にデザイン可能

実装イメージ
<!-- HTML: 構造を自分で組む -->

<div role="combobox"
     aria-expanded="false"
     aria-controls="listbox">
  選択してください ▼
</div>

<ul role="listbox" hidden>
  <li role="option">
    <!-- 自由なレイアウト -->
    <span>🍎</span>
    <span>りんご</span>
    <span>青森県産 / さっぱり甘い</span>
  </li>
  ...
</ul>

<!-- JS: 開閉・キーボード・選択を全部自作 -->
<script>
  cb.addEventListener('click', open)
  cb.addEventListener('keydown', navigate)
  lb.addEventListener('click', select)
</script>
