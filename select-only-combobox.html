2. Select-only Combobox
Sample
選択してください ▼
  • 🍎 りんご青森県産 / さっぱり甘い
  • 🍊 みかん愛媛県産 / ジューシー
  • 🍇 ぶどう山梨県産 / 濃厚な甘み
  • 🍌 バナナフィリピン産 / 栄養たっぷり
  • 🍓 いちご栃木県産 / とちおとめ

ドロップダウンの中身まで自由にデザイン可能

実装イメージ
<!-- HTML: 構造を自分で組む -->

<div role="combobox"
     aria-expanded="false"
     aria-controls="listbox">
  選択してください ▼
</div>

<ul role="listbox" hidden>
  <li role="option">
    <!-- 自由なレイアウト -->
    <span>🍎</span>
    <span>りんご</span>
    <span>青森県産 / さっぱり甘い</span>
  </li>
  ...
</ul>

<!-- JS: 開閉・キーボード・選択を全部自作 -->
<script>
  cb.addEventListener('click', open)
  cb.addEventListener('keydown', navigate)
  lb.addEventListener('click', select)
</script>
3. Editable Combobox（テキスト入力で絞り込み）
Sample

テキスト入力で候補を絞り込み、リストから選択する（DeepL の言語選択と同じパターン）

実装イメージ
<!-- Select-only との違い: input 要素を使う -->

<input
  role="combobox"
  aria-expanded="true"
  aria-controls="listbox"
  aria-autocomplete="list"
/>

<ul role="listbox">
  <li role="option">日本語</li>
  <li role="option">英語</li>
  ...
</ul>

<!-- JS: 入力に応じてフィルタリング -->
<script>
  input.addEventListener('input', () => {
    // 入力テキストで候補を絞り込む
    filtered = items.filter(s =>
      s.includes(input.value)
    )
    renderOptions(filtered)
  })
</script>
